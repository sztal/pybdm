Standard installation (stable)::

    pip install pybdm

Development version installation::

    pip install git+https://github.com/sztal/pybdm.git

Local development::

    git clone https://github.com/sztal/pybdm
    cd pybdm
    pip install --editable .


Supported versions
------------------

Python3.5+ is supported. Tests are run against Linux and OSX, but
Windows should work as well.
